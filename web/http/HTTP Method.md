### HTTP 메서드와 그 역할을 말해보세요



```
HTTP는 자원, 행위, 표현으로 구성되어 있는데. 
그 중 행위에 해당하는 것은 HTTP 메서드라고 표현함. 
+자원(URI), 행위(HTTP Method), 표현(데이터 형식: ex.JSON) 
HTTP 메서드의 역할은 아래와 같음 • 주요 
1. GET: 리소스 조회 
2. POST: 요청 데이터 처리, POST의 결과로 새로운 리소스 생성 안될수도 있음, GET으로 body 넘겨주고 싶은데 서버가 지원 안할 때
3. PUT: 리소스를 대체, 해당 리소스가 없으면 생성 
4. PATCH: 리소스 부분 변경 
5. DELETE: 리소스 삭제 

• 비 주요 
1. HEAD: GET과 동일하지만메시지 부분을 제외하고, 상태 줄과 헤더만 반환 
2. OPTIONS: 대상 리소스에 대한 통신 기능 옵션(메서드)을 설명(주로 CORS에서 사용)

- CONNECT, TRACE 거의 안씀
```
